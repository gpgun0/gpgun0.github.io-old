# NPM

## Node.JS 업그레이드 방법

1. 현재 node 버전 확인

```bash
node -v
```

2. npm 캐시 제거

```bash
npm cache clean -f
```

3. Node.js 버전 관리 모듈 'n' 설치

```bash
npm install -g n
```

4. 원하는 버전 설치

```bash
n stable    # 안정
n lts       # Long Term Support
n latest    # 최신
n 14.15.4   # Specific
```
